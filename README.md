# Line Balancing Analysis Tool
**Optimasi Lini Produksi Berbasis Heuristik**

Aplikasi berbasis web ini dirancang untuk menganalisis dan mengoptimalkan keseimbangan lintasan produksi (Assembly Line Balancing) menggunakan pendekatan matematis. Proyek ini disusun sebagai pemenuhan Tugas Besar (Pengganti UAS) mata kuliah Logika Pemograman.

## üöÄ Fitur Utama
- **CRUD Project Management**: Kelola data proyek lini produksi secara sistematis.
- **Dual Analysis Mode**: Analisis berdasarkan *Actual Bottleneck* atau *Target Takt Time*.
- **Industrial Metrics**: Perhitungan otomatis untuk *Line Efficiency* (LE), *Balance Delay* (BD), dan *Smoothness Index* (SI).
- **Data Visualization**: Grafik distribusi beban kerja interaktif menggunakan Chart.js.
- **Modern UI**: Antarmuka profesional dengan tipografi Inter font.
